# Interfaces
インターフェイスとはプログラムの2つの部分の間の契約のことです.  
あるサービスの供給者とそのサービスの利用者に何を期待するかを正確に記述することが重要です.  
優れた(わかりやすい, 効率的な使用を促す, エラーを起こさない, テストをサポートするなど)インターフェイスを持つことは,  
おそらくコードの構成において最も重要なことです.
